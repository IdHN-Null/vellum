지역·리본 문구·자유 그림까지, 세계관을 지도 위에 그대로 담습니다.

Start

세 걸음이면 충분합니다.

Vellum의 흐름은 단순합니다 — 만들고, 다듬고, 잇는 것.

1

만들기

지형 탭에서 주사위를 굴려, 마음에 드는 대륙이 나올 때까지 생성합니다.

2

다듬기

브러시로 지형을 손보고, 스타일과 색·헤칭을 취향껏 조절합니다.

3

잇기

마커를 찍고 노트를 연결해, 세계관을 하나의 지도로 엮습니다.

설치

압축을 풀고 main.js · manifest.json · styles.css를 확인합니다.
볼트의 .obsidian/plugins/vellum/에 복사합니다. (Windows는 동봉된 install.ps1로 한 번에)
💡 스크립트 오류 시: "not digitally signed" 오류로 설치가 안 된다면, 다운로드 파일의 보안 차단 때문입니다. PowerShell에서 Unblock-File -Path .\install.ps1을 입력해 차단을 풀고 다시 시도하세요.
설정 → 커뮤니티 플러그인에서 Vellum을 켭니다.
명령 팔레트 → 샘플팩 설치로 예제 지도를 받아 시작합니다.

이전에 Fantasy Map Studio를 쓰셨나요? 지도 파일(.fmap)은 그대로 열립니다. 새로 설치한 뒤 기존 플러그인은 비활성화하세요 — 둘 다 켜져 있으면 충돌할 수 있습니다.
