Install

Unzip and find main.js · manifest.json · styles.css.
Copy them into your vault's .obsidian/plugins/vellum/. (On Windows, the bundled install.ps1 does it in one go.)
Enable Vellum under Settings → Community plugins.
Command palette → 샘플팩 설치 (install sample pack) gives you an example map to start from.

Coming from Fantasy Map Studio? Your map files (.fmap) open as-is. After installing, disable the old plugin — running both may conflict.
